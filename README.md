# 実行方法

## 1. リポジトリのクローン

```bash
https://github.com/mila411/django-jwt-token.git
```

---

## 2. 実行

### 2-1. パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2-2. マイグレート

```bash
python manage.py migrate
```

### 2-3. 開発サーバ起動

```bash
python manage.py runserver
```
